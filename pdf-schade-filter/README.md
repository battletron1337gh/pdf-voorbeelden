# PDF Schade Filter

Een web-based tool om automatisch schade-regels uit onderhoudshistorie PDF's te filteren.

## 🚀 Snel Starten

1. Open `index.html` in een moderne browser
2. Upload je PDF met onderhoudshistorie
3. De tool analyseert automatisch alle regels
4. Download de gefilterde PDF zonder schade-regels

## 🛠️ Features

- **Automatische herkenning** van schade-trefwoorden
- **OCR ondersteuning** voor gescande PDF's
- **Preview** van gevonden regels
- **Statistieken** over schade vs onderhoud
- **Eenvoudige download** van gefilterde PDF

## 📝 Herkende Schade Trefwoorden

- schade, botsing, collision
- reparatie, herstel, damage
- ongeval, accident, ramp
- aanrijding, aanvaring, impact
- plaatwerk, lakwerk

## 🔧 Technische Details

- Gebruikt `pdf-lib` voor PDF manipulatie
- Gebruikt `tesseract.js` voor OCR (tekstherkenning)
- 100% client-side - geen server nodig
- Werkt offline na eerste load
